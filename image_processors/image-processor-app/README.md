# Image Processor App

This project is a simple image processing application built using Python's Tkinter library. It allows users to load an image, apply a specified number of iterations of sharpening and blurring effects, and save the processed image.

## Features

- Load an image from your file system.
- Specify the number of iterations for sharpening and blurring.
- Preview the processed image in the GUI.
- Save the processed image to your file system.

## Requirements

To run this application, you need to have Python installed along with the following packages:

- Pillow
- tkinter

You can install the required packages using pip. Make sure to include the following in your `requirements.txt`:

```
Pillow
```

## Installation

1. Clone the repository or download the source code.
2. Navigate to the project directory.
3. Install the required packages:

   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the application:

   ```
   python src/main.py
   ```

2. Use the GUI to load an image.
3. Specify the number of iterations for sharpening and blurring.
4. Click the "Process" button to apply the effects.
5. Preview the processed image.
6. Click the "Save" button to save the processed image.
